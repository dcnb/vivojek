About


The VIVO Explorer! is developed and maintained by the University of Idaho Library to help facilitate the exploration of data within its VIVO discovery tool.

VIVO is a semantic web application implemented across the globe as a way to describe and connect the research and researchers that occurs within and across institutions. The University of Idaho has been implementing our own instance of VIVO over several years now, and VIVO Explorer! queries that data to create visualizations and reports.

More technically speaking, VIVO Explorer! accesses the VIVO RDF dataset using the W3C recommended SPARQL Query Language over HTTP and then uses JavaScript and jQuery on the client browser to generate research reports and narratives for campus organizations. The application also features additional visualizations created using the d3.js JavaScript Visualization Library.

By building this application, we hope to encourage exploration of the university's scholarly and creative activity both internally and externally, as well as to increase interest in VIVO as a reporting and discovery tool for the University of Idaho.

Questions about this application can be directed to Devin Becker @ dbecker@uidaho.edu.
